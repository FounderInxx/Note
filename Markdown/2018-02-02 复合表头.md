# 表头复合的表格
`table` `Founder`
-------------------
> * 先创建表格，后存入数据
### 1. 源代码 `静态页面`
```
<!DOCTYPE html>
<html>
<head>
	<meta charset="utf-8" />
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<title>Page Title</title>
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<link rel="stylesheet" type="text/css" href="css/bootstrap.css" />
	<link rel="stylesheet" type="text/css" href="css/font-awesome.css" />
	<style type="text/css">
		th,td{
			text-align: center;
			overflow: hidden;
		}
	</style>
</head>
<body>
	<div class="container-fluid">
		<div class="row">
			<div class="col-xs-12 col-sm-12">
				<table class="table table-bordered" id="beat">
					<thead>
						<tr>
							<th>00:45</th>
							<th colspan="2" >now</th>						
						</tr>
						<tr>
							<th>No.</th>
							<th>时刻</th>
							<th>日期</th>
						</tr>
					</thead>
					<tbody>
						<tr>
							<th scope="row">1</th>
							<td scope="row">
								<code>8:10</code>
							</td>
							<td scope="row" nowrap>  <!-- 文字不换行 -->
								<code>2018-01-31</code>
							</td>							
						</tr>
					</tbody>
				</table>
			</div>
		</div>
	</div>
	<script src="js/jquery-2.1.1.min.js"></script>
	<script src="js/bootstrap.js"></script>	
	<script type="text/javascript">
		$(function(){
			var x = 10;
			var y = 8;
			var mainhead = '<th colspan="2" >时刻</th>';
			var infohead = '<th>时刻</th><th>日期</th>';
			var xdata = '<th scope="row"><code>8:10</code></th><td scope="row">0</td>';
			for(var i = 0; i < x; i++) {
				//一级表头
				$("#beat thead tr:eq(0)").append(mainhead);
				//二级表头
				$("#beat thead tr:eq(1)").append(infohead);
				//行
				$("#beat tbody tr").append(xdata);				
			}
			for(var j = 0;j < y;j++){
				//列
				$("#beat tbody").append($("#beat tbody tr:eq(0)").clone()); //克隆方法
				//序号
				$("#beat tbody tr:eq("+(j+1)+") th:eq(0)").text(j+2);
			}

		})
	</script>
</body>
</html>
```
